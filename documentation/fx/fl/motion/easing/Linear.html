Class LinearMethods
 
Packagefl.motion.easing
Classpublic class Linear

The Linear class defines easing functions to implement non-accelerated motion with ActionScript animations. Its methods all produce the same effect, a constant motion. The various names easeIn, easeOut and so on, are provided in the interest of polymorphism.
